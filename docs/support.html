Help for Blip: Barcode QR Keyboard.

Need help or want to report a scan issue? Use the in-app feedback button or email blip@antoniobeslic.com.

Setup

  1. Open Blip.
  2. Tap Open Keyboard Settings.
  3. In iOS Settings, tap Keyboards.
  4. Enable Blip.
  5. Enable Allow Full Access so the keyboard and scanner app can share scan results.
  6. Return to Blip and tap Check Again.

Scanning from another app

  1. Open Safari, Chrome, Edge, Firefox, Brave, or another configured target app.
  2. Tap the text field where the scan should be inserted.
  3. Switch to the Blip keyboard using the iOS keyboard button.
  4. Tap Scan.
  5. Scan the barcode or QR code.
  6. Blip returns to the configured target app and inserts the scan into the active text field.

Troubleshooting

  • If Blip does not appear as a keyboard, check iOS Settings > Apps > Blip > Keyboards.
  • If scans do not insert, confirm Allow Full Access is enabled for the Blip keyboard.
  • If the wrong app opens after scanning, check Settings > Return Target in Blip.
  • For the cleanest employee workflow, remove unused keyboards from iOS Keyboard settings.
